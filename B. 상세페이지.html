배달장부 연동하기

스로틀 앱과 배달장부를 연동하여 정산 내역을 실시간으로 관리해 보세요.

1

지갑 메뉴 진입

메인 화면의 [지갑] 버튼을 클릭합니다.

2

플랫폼 추가

지갑 화면 우측 상단의 [플랫폼 추가]를 클릭합니다.

3

플랫폼 선택

화면 하단에 나타난 바텀시트에서 [배달 장부]를 클릭합니다.

4

즉시 연동

계정 연동을 위해 [즉시연동]을 클릭합니다.

5

연동 완료

연동이 완료되었습니다

핸드폰 번호 변경 시 연동 방법

핸드폰 번호 변경 시, 스로틀 앱과 재연동을 시도하세요.

1

지갑 메뉴 진입

메인 화면의 [지갑] 버튼을 클릭합니다.

2

플랫폼 추가

지갑 화면 우측 상단의 [플랫폼 추가]를 클릭합니다.

3

다른 번호로 본인 인증

계정 연동을 위해 [다른 번호로 본인 인증]을 클릭합니다.

4

본인 인증

본인 인증을 위해 정보 입력 후 [인증하기]를 클릭합니다.

5

연동 완료

연동이 완료되었습니다

수익금 출금하기

연동된 배달장부의 수익금을 클릭 몇 번으로 내 계좌에 입금합니다.

1

출금 버튼 클릭

지갑 화면 배달장부 카드 하단의 [출금] 버튼을 누릅니다.

2

출금 금액 입력

출금하려는 금액을 입력 후 하단의 [다음] 버튼을 누릅니다.

3

입금 예정 금액 확인

수수료 확인 후 최종 입금 예정 금액을 확인 후 [출금하기] 버튼을 누릅니다.

4

2차 비밀번호 입력

2차 비밀번호 입력하면 [출금 완료] 됩니다.

상세내역 조회하기

내 정산금의 세부 항목과 일자별 입출금 기록을 확인하세요.

1

배달장부 카드 클릭

지갑 메인에서 배달장부 영역을 클릭해 상세 화면으로 이동합니다.

2

배달장부 상세 내역 조회

상세 내역에서 입출금 현황 및 각 정산금확인이 가능합니다.

차감요청 승인하기

협력사에서 보낸 차감 요청을 확인하고 정산금에서 공제되도록 승인합니다.

1

승인 배너 클릭

상세 화면 상단의 [차감 요청 승인] 배너를 클릭합니다.

2

차감 요청 수락

차감 요청 내역을 확인 후 [수락] 버튼을 클릭합니다.

차감현황 조회하기

진행 중인 리스비, 보험료 등 차감 항목의 납부 현황을 확인하세요.

1

차감 관리 진입

상세 화면 중앙의 [차감 관리] 버튼을 누릅니다.

2

차감 리스트 확인

차감 진행 중인 리스트 확인이 가능합니다. 상세 내역은 진행중인 [카드] 를 클릭합니다.

3

차감 내역 상세 확인

차감되고 있는 카드의 [미납 및 차감 완료] 내역 확인이 가능합니다.

계좌 수정하기

정산금을 입금받을 은행 계좌 정보를 안전하고 간편하게 변경할 수 있습니다.

1

더보기 아이콘 클릭

지갑 메인 화면에서 배달장부 카드 왼쪽의 더보기(⋮) 아이콘을 누릅니다.

2

계좌 수정 메뉴 선택

메뉴 목록에서 [계좌 수정]을 클릭합니다. (계좌 삭제나 연동 해제도 이곳에서 가능합니다.)

3

은행 및 계좌 정보 입력

계좌 등록 화면에서 은행을 선택하고, 본인 명의의 계좌주 성함과 계좌번호를 입력합니다.

4

정보 등록 완료

정보를 모두 입력했다면 하단의 [등록] 버튼을 클릭하여 수정을 마무리합니다.

도움말

  • • 보안을 위해 본인 명의의 계좌만 등록 가능합니다.
  • • 계좌 정보가 불일치할 경우 출금이 거절될 수 있습니다.
